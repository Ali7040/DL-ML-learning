# ML & DL Resources Compilation 📚

A curated list of the best resources for mastering Machine Learning and Deep Learning.

---

## 📖 Books

### Mathematics
1. **"Mathematics for Machine Learning"** by Deisenroth, Faisal, Ong
   - Free PDF available
   - Covers Linear Algebra, Calculus, Probability
   - Perfect for beginners

2. **"Linear Algebra Done Right"** by Sheldon Axler
   - Deeper dive into linear algebra
   - Conceptual understanding

3. **"Probability and Statistics"** by DeGroot & Schervish
   - Statistical foundations
   - Covers all distributions

### Machine Learning
4. **"Pattern Recognition and Machine Learning"** by Christopher Bishop
   - Comprehensive ML textbook
   - Mathematical rigor
   - Bayesian perspective

5. **"The Elements of Statistical Learning"** by Hastie, Tibshirani, Friedman
   - Classic ML book
   - Free PDF
   - Statistical approach

6. **"Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow"** by Aurélien Géron
   - Practical approach
   - Code examples
   - Perfect for practitioners

7. **"Machine Learning Yearning"** by Andrew Ng
   - Free online
   - Practical advice
   - Building ML projects

### Deep Learning
8. **"Deep Learning"** by Goodfellow, Bengio, Courville
   - The DL Bible
   - Comprehensive coverage
   - Free online

9. **"Deep Learning with PyTorch"** by Eli Stevens
   - PyTorch from basics to advanced
   - Step-by-step implementations

10. **"Dive into Deep Learning"** (d2l.ai)
    - Free interactive book
    - Code with every concept
    - Multiple frameworks

11. **"Neural Networks and Deep Learning"** by Michael Nielsen
    - Free online
    - Intuitive explanations
    - Great for beginners

### Specialized Topics
12. **"Reinforcement Learning: An Introduction"** by Sutton & Barto
    - RL Bible
    - Free online
    - Theory and algorithms

13. **"Natural Language Processing with Transformers"** by Hugging Face team
    - Modern NLP
    - Transformers in detail
    - Practical code

## 💡 Tips for Using Resources

1. **Don't try to consume everything** - Pick 2-3 resources per topic
2. **Active learning** - Implement while reading
3. **Spaced repetition** - Review concepts regularly
4. **Multiple perspectives** - Same topic from different sources
5. **Community** - Join discussions, ask questions
6. **Teach others** - Best way to solidify understanding
7. **Stay current** - But don't chase every new paper
8. **Build while learning** - Projects over passive consumption

---

**Remember:** The best resource is the one you actually use! Start small, stay consistent. 📚✨
