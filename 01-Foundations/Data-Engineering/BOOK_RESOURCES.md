# Data Engineering Books & Resources 📚

Curated list of the best resources for learning data engineering in the context of ML/DL.

---

## 📖 Essential Books

### 1. **"Python for Data Analysis"** by Wes McKinney ⭐
- **Author:** Creator of Pandas
- **Why:** THE reference for Pandas
- **Level:** Beginner to Intermediate
- **Topics:**
  - NumPy fundamentals
  - Pandas in-depth
  - Data loading and storage
  - Data cleaning and preparation
  - Data wrangling
  - Time series analysis

**Where to Get:**
- [O'Reilly](https://www.oreilly.com/library/view/python-for-data/9781491957653/)
- Amazon
- Free code examples: [GitHub](https://github.com/wesm/pydata-book)

**How to Use:**
- Chapters 4-7 for Pandas mastery
- Chapter 7 specifically for data cleaning
- Work through all examples

---

### 2. **"Feature Engineering for Machine Learning"** by Alice Zheng & Amanda Casari
- **Why:** Best book on feature engineering
- **Level:** Intermediate
- **Topics:**
  - Fundamental techniques
  - Text data feature engineering
  - Image features
  - Dimensionality reduction
  - Category encoding

**Where to Get:**
- [O'Reilly](https://www.oreilly.com/library/view/feature-engineering-for/9781491953235/)
- Amazon

**How to Use:**
- Read chapters 1-4 during Week 6
- Implement all examples
- Apply to your projects

---

### 3. **"Data Wrangling with Python"** by Jacqueline Kazil & Katharine Jarmul
- **Why:** Practical data cleaning focus
- **Level:** Beginner
- **Topics:**
  - Data acquisition
  - Cleaning messy data
  - Working with APIs
  - Web scraping
  - Data storage

**Where to Get:**
- [O'Reilly](https://www.oreilly.com/library/view/data-wrangling-with/9781491948804/)

---

### 4. **"Pandas Cookbook"** by Theodore Petrou
- **Why:** Recipe-based learning
- **Level:** All levels
- **Topics:**
  - 100+ pandas recipes
  - Real-world scenarios
  - Performance tips
  - Best practices

**Where to Get:**
- [Packt](https://www.packtpub.com/product/pandas-cookbook/9781784393878)

---
